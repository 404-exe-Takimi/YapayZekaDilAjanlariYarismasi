# Yapay Zeka Dil Ajanları Yarışması - 404.exe Takımı

## Proje Özeti
BDDK listesindeki katılım bankalarının kampanya ve ürün verilerini anlık analiz eden, dış kaynaklara bağımlılığı olmayan ve tamamen kurum içi (on-premise) sunucularda güvenle çalışabilecek açık kaynaklı bir yapay zeka dil ajanı projesidir.

## Gerekli Bağımlılıklar (Dependencies)
- Python 3.10+
- Gerekli kütüphaneler proje içerisindeki kod dosyalarında belirtilmiştir.

## Çalıştırma Adımları
Proje geliştirme aşamasındadır. Ana script'i çalıştırmak için:
python tum_kampanyalar.py

## Veri Seti Bağlantısı
Projede kullanılan temel finansal kuruluş listesi için BDDK resmi internet sitesindeki Katılım Bankaları verileri referans alınmıştır.
